TO START PROJECT
1) Create executable with project4.cc (one has been provided as project4)
2) Run "player project4.cfg" in terminal
3) Open another terminal and run "./project4 2"
	-This runs the robot leader
4) Open another terminal and run "./project4 1"
	-This runs the task manager and robot followers
	-input the following commands when asked
		line -11.05 0.09
		diamond 0.05 0.09
		line 10.05 -0.09
		stop
5) It should start running the player simulation
